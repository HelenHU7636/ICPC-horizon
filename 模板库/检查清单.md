## 2D dp

- [ ] 返回下标：`dp[n][m]` 还是 `dp[n-1][m-1]`？
- [ ] 初始值给的够吗？第0行第0列有无显式初始化？
- [ ] 循环上界：`i <= n` 还是 `i < n`？有没有越界？
- [ ] dp[i][j]`+1` 有没有误写成 `++`（会修改原值且返回修改后的值）？
- [ ] 变量名有没有拼错 / 索引混用（如 `dp[i][j]` 写成 `dp[j][i]`）？

## 背包专项

- [ ] 0-1 背包：重量维度**倒序**遍历了吗？
- [ ] 完全背包：重量维度**正序**遍历了吗？
- [ ] 求方案数时返回值是否会溢出 int？

## 双序列 DP 专项

- [ ] 1-indexed：`s[i-1]`、`t[j-1]` 取字符，不是 `s[i]`？
- [ ] `*` 的语义：通配符（独立匹配任意串）vs 正则（依附前一字符）？

## 区间 DP 专项

- [ ] 外层循环是区间长度 `len`，不是左端点 `i`？
- [ ] `k` 的枚举范围：`[i, j-1]` 还是 `[i+1, j]`，边界有没有漏？

## 图 / 树专项（备用）

- [ ] 邻接表建边有没有漏反向边？
- [ ] vis 数组有没有初始化？
